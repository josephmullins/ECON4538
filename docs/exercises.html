Problem 1: Income Process Identification

Consider the income process:

\[y_{it} = \alpha_i + \xi_{it} + \varepsilon_{it}\]

where \(\xi_{it} = \rho \xi_{it-1} + \eta_{it}\) and \(\eta_{it}, \varepsilon_{it}\) are iid with \(\mathbb{V}[\alpha_i] = \sigma_\alpha^2\), \(\mathbb{V}[\eta_{it}] = \sigma_\eta^2\), \(\mathbb{V}[\varepsilon_{it}] = \sigma_{\varepsilon,t}^2\).

Use population moments to argue that the parameters \(\sigma_\alpha^2, \rho, \sigma_\eta^2, \sigma_{\varepsilon,t}^2\) can be identified with panel data. Note that \(\sigma^2_{\varepsilon,t}\) is specific to the age \(t\).

Solution

Variance of income:

\[\mathbb{V}[y_{it}] = \sigma_\alpha^2 + \mathbb{V}(\xi_{it}) + \sigma_{\varepsilon,t}^2\]

\[\mathbb{V}[y_{it+1}] = \sigma_\alpha^2 + \rho^2 \mathbb{V}(\xi_{it}) + \sigma_\eta^2 + \sigma_{\varepsilon,t}^2\]

Covariances:

\[\mathcal{C}(y_{it}, y_{it+1}) = \sigma_\alpha^2 + \rho \mathbb{V}(\xi_{it})\]

\[\mathcal{C}(y_{it}, y_{it+2}) = \sigma_\alpha^2 + \rho^2 \mathbb{V}(\xi_{it})\]

\[\mathcal{C}(y_{it}, y_{it+3}) = \sigma_\alpha^2 + \rho^3 \mathbb{V}(\xi_{it})\]

Identification:

\[\rho = \frac{\mathcal{C}(y_{it}, y_{it+3}) - \mathcal{C}(y_{it}, y_{it+2})}{\mathcal{C}(y_{it}, y_{it+2}) - \mathcal{C}(y_{it}, y_{it+1})} = \frac{(\rho^3 - \rho^2) \mathbb{V}(\xi_{it})}{(\rho^2 - \rho) \mathbb{V}(\xi_{it})}\]

\[\mathbb{V}(\xi_{it}) = \frac{\mathcal{C}(y_{it}, y_{it+2}) - \mathcal{C}(y_{it}, y_{it+1})}{\rho^2 - \rho}\]

\[\sigma_\alpha^2 = \mathcal{C}(y_{it}, y_{it+1}) - \rho \mathbb{V}(\xi_{it})\]

\[\sigma_{\varepsilon,t}^2 = \mathbb{V}[y_{it}] - \sigma_\alpha^2 - \mathbb{V}(\xi_{it}) \quad \text{for each } t\]

\[\sigma_\eta^2 = \mathbb{V}[y_{it+1}] - \sigma_\alpha^2 - \rho^2 \mathbb{V}(\xi_{it}) - \sigma_{\varepsilon,t}^2\]

Done.

Problem 3: Income Process Identification

Consider the income process:

\[y_{it} = \alpha_i + \xi_{it} + \varepsilon_{it} + \theta \varepsilon_{i,t-1}\]

where \(\xi_{it} = \rho \xi_{it-1} + \eta_{it}\), and \((\eta_{it}, \varepsilon_{it})\) are iid with \(\mathbb{V}[\alpha_i] = \sigma_\alpha^2\), \(\mathbb{V}[\varepsilon_{it}] = \sigma_{\varepsilon}^2\), \(\mathbb{V}[\eta_{it}] = \sigma_\eta^2\).

Use population moments to identify the parameters of the income process.

Solution

Variance of income:

\[\mathbb{V}[y_{it}] = \sigma_\alpha^2 + \mathbb{V}(\xi_{it}) + (1+\theta^2) \sigma_{\varepsilon}^2\]

\[\mathcal{C}(y_{it}, y_{it+1}) = C_1 = \sigma_\alpha^2 + \rho \mathbb{V}(\xi_{it}) + \theta \sigma_{\varepsilon}^2\]

\[\mathcal{C}(y_{it}, y_{it+2}) = C_2 = \sigma_\alpha^2 + \rho^2 \mathbb{V}(\xi_{it})\]

\[\mathcal{C}(y_{it}, y_{it+s}) = C_s = \sigma_\alpha^2 + \rho^s \mathbb{V}(\xi_{it}) \quad \forall \, s > 1\]

Identification:

\[\rho = \frac{C_4 - C_3}{C_3 - C_2} = \frac{\rho^4 - \rho^3}{\rho^3 - \rho^2}\]

\[\mathbb{V}(\xi_{it}) = \frac{C_3 - C_2}{\rho^3 - \rho^2}, \quad \sigma_\alpha^2 = C_2 - \rho^2 \mathbb{V}(\xi_{it})\]

\[\frac{(1+\theta^2)}{\theta} = \frac{\mathbb{V}(y_{it}) - \sigma_\alpha^2 - \mathbb{V}(\xi_{it})}{C_1 - \sigma_\alpha^2 - \rho \mathbb{V}(\xi_{it})}\]

\(\Rightarrow\) \(\theta\) is solution to quadratic.

\[\Rightarrow \sigma_{\varepsilon}^2 = \frac{\mathbb{V}(y_{it}) - \sigma_\alpha^2 - \mathbb{V}(\xi_{it})}{(1+\theta^2)}\]

Done.

Problem 4: One-Period Model with Lognormal Income

Consider the following one period model:

\[C_i = Y_i = \exp(\alpha_i + \varepsilon_i)\]

where \(\varepsilon_i \sim \mathcal{N}(0, \sigma_{\varepsilon}^2)\), \(\alpha_i \sim \mathcal{N}(\mu, \sigma_\alpha^2)\). Suppose that \(\alpha_i\) is known to each individual and \(\varepsilon_i\) is income risk. Assume log utility s.t. \(V(\alpha_i) = \mathbb{E}[\log(C_i) | \alpha_i]\).

Part (a)

Show that \(V(\alpha_i) = \alpha_i\).

Solution

\[V(\alpha_i) = \mathbb{E}[\log(C_i) | \alpha_i] = \mathbb{E}[\alpha_i + \varepsilon_i | \alpha_i] = \alpha_i\]

Part (b)

Suppose a planner solves:

\[\max_{\{C_i\}_{i\in[0,1]}} \int \log(C_i) di \quad \text{s.t.} \quad \int C_i di = \int \exp(\alpha_i + \varepsilon_i)di\]

Show that the solution is

\[C_i^* = C^* = \exp(\mu + \frac{1}{2}(\sigma_{\varepsilon}^2 + \sigma_\alpha^2))\]

Hint: if \(Z \sim \mathcal{N}(\mu, \sigma^2)\), \(\mathbb{E}[\exp(Z)] = \exp(\mu + \frac{1}{2}\sigma^2)\).

Solution

Since \(\alpha_i + \varepsilon_i \sim \mathcal{N}(\mu, \sigma_\alpha^2 + \sigma_{\varepsilon}^2)\), \(\int\exp(\alpha_{i}+\varepsilon_{i})di = \exp(\mu + \frac{1}{2}(\sigma_{\varepsilon}^2 + \sigma_\alpha^2))\).

Planner’s FOC: \(\frac{1}{C_i} = \lambda \quad \forall \, i\)

where \(\lambda\) is multiplier on resource constraint. Hence \(C_i = C^* \quad \forall \, i\).

Plugging into resource constraint gives

\[C^* = \exp(\mu + \frac{1}{2}(\sigma_{\varepsilon}^2 + \sigma_\alpha^2))\]

Part (c)

Show that the certainty equivalent \(\overline{C}(\alpha_i)\) that delivers \(V(\alpha_i)\) is \(\overline{C}(\alpha_i) = \exp(\alpha_i)\).

Solution

\(\overline{C}(\alpha_i)\) solves

\[\log(\overline{C}(\alpha_i)) = V(\alpha_i) = \alpha_i\]

\[\Rightarrow \overline{C}(\alpha_i) = \exp(\alpha_i)\]

Part (d)

Show that \(\overline{C} = \int \overline{C}(\alpha_{i})di = \exp(\mu + \frac{1}{2}\sigma^2_{\alpha})\)

Note

Since \(\alpha_i \sim \mathcal{N}(\mu, \sigma_\alpha^2)\),

\[\overline{C} = \int \overline{C}(\alpha_i) di = \exp(\mu + \frac{1}{2}\sigma_\alpha^2)\]

Part (e)

Show that the planner’s objective is \(\mu\) in the baseline and \(\mu + \frac{1}{2}(\sigma_{\varepsilon}^2 + \sigma_\alpha^2)\) at the optimum.

Solution

Baseline: \(W = \int \log(C_i) = \int \log(\exp(\alpha_i + \varepsilon_i)) di = \mathbb{E}[\alpha_i + \varepsilon_i] = \mu\)

\[W^* = \log(C^*) = \mu + \frac{1}{2}(\sigma_{\varepsilon}^2 + \sigma_\alpha^2)\]

Part (f)

Decompose \(W^* - W\) into two components:

  1. \(W^* - \log(\overline{C})\)
  2. \(\log(\overline{C}) - W\)

What is (1)? What is (2)? Which term represents the gains from insurance?

Solution

  1. \(W^* - \log(\overline{C}) = \frac{1}{2}\sigma_{\varepsilon}^2 \leftarrow\) Insurance

  2. \(\log(\overline{C}) - W = \frac{1}{2}\sigma_\alpha^2 \leftarrow\) Redistribution

Part (g)

Interpret this result.

Interpretation

\(\sigma_{\varepsilon}^2\) measures the variance in income coming from idiosyncratic risk that individuals face. Insurance eliminates this uncertainty.

\(\sigma_\alpha^2\) measures the variance in income coming from permanent differences across individuals (ex-ante inequality). Redistribution addresses this component.

Problem 5: Two-Period Becker-Tomes Model

Consider the 2-period Becker-Tomes model:

\[\max_{C_1, C_2, X, A} U(C_1) + \beta U(C_2)\]

\[\text{s.t.} \quad C_1 + \psi X + A / (1+r) \leq \Omega + T / (1+r)\]

\[C_2 = A + wz X^\delta - T, \quad A \geq 0\]

where \(T\) is a lump-sum tax paid by generation 2 to generation 1.

Part (a)

Show that if a household is not constrained, the value of \(T\) does not affect any allocations of \(C_1, C_2, X\) at the optimum. (Hint: try combining budget constraints).

Solution

If \(A > 0\) at the optimum, can combine to get:

\[C_1 + C_2/(1+r) \leq \Omega + wz X^\delta/(1+r) \quad (*)\]

s.t. household solves:

\[\max_{C_1, C_2} U(C_1) + \beta U(C_2) \quad \text{s.t.} \quad (*)\]

Hence \(T\) does not feature.

Part (b)

Show that for constrained households:

\[U'(C_1) = \frac{\delta wz X^{\delta-1}}{(1+r)} U'(C_2) \quad \text{and} \quad (1+r) \leq \frac{\delta wz X^{\delta-1}}{\psi}\]

Solution

When \(A = 0\), solve:

\[\max_{C_1, C_2} U(\Omega + T/(1+r) - \psi X) + \beta U(wz X^\delta - T)\]

get

\[U'(\Omega + T/(1+r) - \psi X) = \beta \delta wz X^{\delta-1} U'(wz X^\delta - T)\]

\[\Leftrightarrow U'(C_1) \psi = \beta \delta wz^{\delta-1} U'(wz X^\delta - T)\]

Note that the marginal return (in assets) is

\[-\frac{1}{1+r} U'(C_1) + \beta U'(C_2) \leq 0 \quad \Leftrightarrow (1+r) \leq \frac{\delta wz X^{\delta-1}}{\psi}\]

(if \(> 0\), then household should optimally increase \(A\) above 0).

Part (c)

Consider a marginal increase in \(T\). What happens to the welfare of constrained vs unconstrained households?

Solution

If \(T \nearrow\) marginally, change in welfare is

\[u'(C_1) \frac{1}{(1+r)} - \beta u'(C_2) > 0\]

for constrained households, where inequality follows from (b).

Since allocations of constrained households are unaffected by \(T\) \(\Rightarrow\) no welfare effect.

Note: not clear if welfare of children \(\nearrow\) in this intervention.

Problem 6: Two-Period Model with Progressive Taxation

For the same 2-period model, suppose budget constraints are now:

\[C_1 + A/(1+r) + \psi X \leq \lambda_{1} \Omega^{1-\tau}\]

\[C_2 = A + \lambda_2 (wz X^\delta)^{1-\tau}, \quad A \geq 0\]

where \(\lambda_1\) and \(\lambda_2\) are chosen to balance budget in each period.

Part (a)

Write the FOC for unconstrained households.

Solution

For unconstrained:

\[(1+r) = \frac{\lambda_2 (1-\tau) \delta(w z)^{1-\tau} X^{\delta(1-\tau)-1}}{\psi}\]

Part (b)

Recall that the planner would set, if possible, \((1+r) = \frac{\delta wz X^{\delta-1}}{\psi}\) in each household \(i\). Can the equilibrium with taxation be efficient?

Solution

Comparing to the condition \((1+r) = \frac{\delta wz X^{\delta-1}}{\psi}\),

taxation distorts the optimal allocation of financials \(\Rightarrow\) human capital \(\Rightarrow\) not efficient.

Part (c)

Explain, in words, the efficiency cost of progressive taxation in this model. (Hint: compare the household’s FOC to the planner’s).

Solution

Redistribution reduces the incentives for parents to invest. \(\Rightarrow\) An equity-efficiency tradeoff.

Part (d)

Assuming log utility, solve for \(X\) when households are constrained. Comment on solution relative to Baseline.

Solution

\[\max_X \log(\lambda_1 \Omega^{1-\tau} - \psi X) + \beta \log(\lambda_2 wz X^\delta)^{1-\tau}\]

\[\Rightarrow \frac{\psi}{\lambda_1 \Omega^{1-\tau} - \psi X} = \frac{\beta (1-\tau) \delta}{\chi}\]

\[\Leftrightarrow X = \frac{1}{\psi} \times \frac{\beta (1-\tau) \delta}{1 + \beta(1-\tau) \delta} \lambda_1 \Omega^{1-\tau}\]

\[\Rightarrow \text{compared to } X = \frac{1}{\psi} \frac{\beta \delta}{1+\beta \delta} \Omega\]

So taxation also distorts investment in unconstrained. This could offset distortion from incomplete markets, but depends on parameters \((\lambda_1, \lambda_2, \tau)\).
